# Angelos Test Plan

## Add task
- Say: "Study chemistry tomorrow at 6"
- Expected: Task appears, due date appears, time appears, app confirms aloud.

## Next task
- Add one task for today and one for tomorrow.
- Expected: Today's task appears as the next task.

## Read tasks
- Say: "Read my tasks"
- Expected: App reads all unfinished tasks aloud.

## Count tasks
- Say: "How many tasks do I have?"
- Expected: App says number of active and completed tasks.

## Complete task
- Say: "Mark chemistry complete"
- Expected: Chemistry task becomes completed.

## Delete task
- Say: "Delete chemistry"
- Expected: Chemistry task is removed.

## Undo delete
- Say: "Undo"
- Expected: Deleted task is restored.

## Edit task
- Click Edit.
- Change task text.
- Expected: Task updates correctly.
